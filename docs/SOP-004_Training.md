# SOP-004 Training and Competency

Document ID: SOP-004  
Version: 1.0  
Effective Date: 2026-04-06

## 1. Purpose

Ensure personnel are trained and competent before performing trial-critical tasks.

## 2. Scope

Applies to staff performing software operations, QA review, validation execution, and incident response.

## 3. Procedure

1. Define role-based training matrix.
2. Deliver initial and periodic refresher training.
3. Assess competency (quiz, supervised run, signoff).
4. Record completion and due dates.
5. Restrict critical task execution for untrained personnel.

## 4. Minimum Training Topics

- GxP basics and data integrity (ALCOA+)
- System operation and SOP adherence
- Security and privacy handling
- Deviation/CAPA and incident reporting
