# SOP-003 Document Control

Document ID: SOP-003  
Version: 1.0  
Effective Date: 2026-04-06

## 1. Purpose

Define how controlled documents are authored, reviewed, approved, versioned, distributed, and retired.

## 2. Scope

Applies to validation, quality, security, and operational documentation used for OncoSML clinical-support workflows.

## 3. Roles

- Document Owner: Creates and updates content.
- QA Reviewer: Verifies compliance and formatting.
- Approver: Grants final approval for use.

## 4. Procedure

1. Draft using controlled template and unique document ID.
2. Submit for QA review and comment resolution.
3. Approve with signature/date and publish controlled version.
4. Track revisions in history table.
5. Obsolete prior version and archive per retention policy.

## 5. Record Retention

- Controlled copies retained for minimum 7 years unless superseded by policy.
